2. nil

  • Objective-C 에서는 포인터를 지원하기 때문에 다음의 4가지를 사용. (Swift에서는 nil만 사용!)
nil Nil NULL NSNull
Object의 부재 Class의 부재 C-Pointer의 부재 NSObject를 상속받는 객체로 collection item을 null로 설정할 필요가 있을때 사용
NSString *temp = nil; Class temp = Nil int *temp = NULL; [temp addObject:[NSNull null]];
if ([temp objectAtIndex:0] == [NSNull null])

3. 타입추론

// 변수
var a = 1;
// 상수
let b = 2;
  • 컴파일 시점에서 변수의 타입을 지정
    let a = 1234 // int 
let b = "test" //String
  • 하지만 아래와같이 사용하면 컴파일 시 어떤타입으로 지정할지 알 수 없음. 컴파일러가 점쟁이는 아니므로 값을 할당하지 않을거면
    let c
  • 아래와 같이 타입을 지정(어노테이션)!
    let c : String

  print("funking swift")
